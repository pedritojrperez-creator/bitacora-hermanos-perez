# 📊 Bitácora Ejecutiva — Hermanos Pérez

**Sistema de gestión de proyecto digital editable en tiempo real con Supabase**

## 🎯 Características

✅ **Completamente editable** — Títulos, descripciones, tareas  
✅ **Sincronización en tiempo real** — Todos los cambios se guardan en Supabase  
✅ **Upload de archivos** — Sube documentos a cada fase  
✅ **Progreso automático** — Calcula el avance al marcar tareas  
✅ **Diseño minimalista** — Blanco, limpio, profesional  
✅ **Responsive** — Funciona en desktop, tablet, mobile e iPad Pro  

## 🚀 Uso Rápido

### Opción 1: Abrir localmente
1. Descarga `index.html`
2. Abre en navegador
3. ¡Listo! Todos los datos se sincronizan con Supabase

### Opción 2: Host en Vercel (Recomendado)
1. Sube este proyecto a GitHub
2. Conecta a [Vercel](https://vercel.com)
3. Deploy automático en `https://tu-dominio.vercel.app`

## 📝 Cómo usar

### Navegar
- Haz clic en cualquier **nodo (1-7)** para expandir la fase
- Los datos se guardan automáticamente

### Editar
- **Títulos y descripciones**: Haz clic y edita directamente
- **Tareas**: Agrega, marca como completadas, elimina
- **Documentos**: Sube archivos que se guardan en Supabase Storage

### Monitorear progreso
- La barra superior muestra el % general
- Las estadísticas se actualizan en tiempo real
- El progreso de cada fase es automático

## 🔗 Tecnologías

- **HTML/CSS/JavaScript** — Frontend puro (sin build)
- **Supabase** — Base de datos + Storage + Auth
- **Vercel** — Hosting automático

## ⚙️ Configuración Supabase

Las tablas ya están creadas:
- `fases` — Las 7 fases del proyecto
- `tareas` — Tareas por fase
- `documentos` — Archivos subidos

**Las credenciales de Supabase ya están en el código** (anon key pública, segura).

## 📱 Compatibilidad

- ✅ Chrome, Firefox, Safari, Edge
- ✅ Windows, Mac, Linux
- ✅ iPhone, Android
- ✅ iPad Pro 13" (optimizado)

## 🔐 Seguridad

- Solo usa la **anon key pública** (no hay credenciales sensibles)
- Row Level Security habilitado en Supabase
- Los datos son de tu proyecto Supabase (no compartidos)

## 📞 Soporte

Si tienes problemas:
1. Verifica que tu proyecto Supabase esté activo
2. Abre la consola del navegador (F12) para ver errores
3. Verifica la conexión a internet
